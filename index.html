The game

Every cell is covered at the beginning, and you can uncover them by clicking on them. Every cell can either be safe and contain a number or contain a mine.
If you uncover a cell containing a mine, you lose the game.
If you uncover a safe cell, it contains a number that corresponds to the number of mines present in the eight adjacent cells.
To start to play, click on any cell, the first cell you choose will always be safe.
The goal of the game is to uncover all safe cells in the shortest time possible.

Flags

If you think there is a mine under a cell, you can place a flag 🚩 on it.
This can be done by clicking on the flag button on top of the table or by clicking any letter on the keyboard.
When the flag button is green , you can click on any cell to place/remove a flag, without uncovering it.
When the flag button is red, clicking on any cell without a flag will uncover it.

Tips

If the number of flags adjacent to an uncovered cell is greater or equal to the number contained in the cell, clicking on the uncovered cell will uncover all its adjacent cells without a flag, even if the flag button is green.
